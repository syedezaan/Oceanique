# Oceanique

A premium seafood restaurant landing page.

## About

A Bootstrap-based landing page designed for a seafood restaurant, guided by a curated visual reference. Features a dark navy and gold color theme with elegant typography for an upscale, premium feel.

## Features

- Dark navy/gold theming with Playfair Display typography
- Scroll-triggered animations using IntersectionObserver
- Custom canvas-based water ripple background effect
- Dark mode toggle
- Floating WhatsApp contact button

## Tech Stack

`HTML` `CSS` `Bootstrap 5`
